# Use `msg.sender` instead of `tx.origin`

### Summary

In the `mint()`, `burn()`, `open()`, `close()`, and `liquidate()` functions, the protocol uses `tx.origin` to identify the user. Using `tx.origin` makes users more vulnerable to phishing attacks.
